LEKCJA 5 – Miejscownik w języku polskim

MIEJSCOWNIK (LICZBA POJEDYNCZA) — ПРОСТОЕ ПРАВИЛО

📍 Когда используется
  • где? -> w pracy
  • где? -> w szkole
  • о чем? -> o pracy
  • после -> po pracy

📌 основные предлоги

w / na / o / po / przy

Rodzaj żeński
(женский род)

-a → -e

  • szkoła -> w szkole
  • sprawa -> w sprawie
  • kobieta -> o kobiecie

после -ca -a → -y

  • praca -> w racy
  • ulica -> na ulicy
  • granica -> na granicy
Rodzaj męski
(мужской род)

чаще всего → -e

  • problem -> o problemie
  • student -> o studencie
  • urzęd -> w urzędzie

иногда → -u

  • dom -> w domu
  • hotel -> w hotelu
  • biur -> w biurze
Rodzaj nijaki (средний род)

чаще всего -o → -e

  • biuro -> w biurze
  • miasto -> w mieście
  • okno -> w oknie

-e / -ie → -u

  • mieszkanie -> w mieszkaniu
  • miejsce -> w miejscu

📌 Самый простой способ понять miejscownik

Задай 2 вопроса:

1️⃣ Есть ли предлог?

w / na / o / po / przy

2️⃣ Это вопрос

gdzie?

или

o czym?

Если да → miejscownik
