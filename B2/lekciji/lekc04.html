LEKCJA 4 – Liczba mnoga w języku polskim

Общее правило

Во множественном числе существительные обычно получают окончания:

-y / -i / -e
1️⃣ Самое простое правило (80% слов)

Очень часто во множественном числе просто добавляется:

-y / -i
  • dokument -> dokumenty
  • student -> studenci
  • problem -> problemy
  • kolega -> koledzy
  • telefon -> telefony
  • sklep -> sklepy

📌 Это самая частая модель.

2️⃣ Женский род

Если слово заканчивается на -a, обычно: -a → -y

  • kobieta -> kobiety
  • sprawa -> sprawy
  • książka -> książki
3️⃣ Средний род

Если слово заканчивается на -o / -e / -ę, обычно: → -a

  • okno -> okna
  • miejsce -> miejsca
  • zadanie -> zadania
  • pytanie -> pytania

4️⃣ Самые важные исключения

Их просто нужно запомнить.

  • człowiek -> ludzie
  • dziecko -> dzieci
  • rok -> lata
  • oko -> oczy
  • ucho -> uszy
  • nauczyciel -> nauczyciele
Когда слово заканчивается на -nik, во множественном числе часто: -nik → -nicy
  • pracownik -> pracownicy
  • rolnik -> rolnicy
  • urzędnik -> urzędnicy
Когда слово заканчивается на -iel / -el, во множественном числе часто: -e
  • nauczyciel -> nauczyciele
  • przyjaciel -> rprzyjaciele
  • obywatel -> obywatele
