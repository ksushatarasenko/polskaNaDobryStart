2️⃣ GRAMATYKA (структурно)

📌 1. Dopełniacz po przeczeniu

Если есть отрицание → объект переходит в dopełniacz.

Mam czas.

Nie mam czasu.

Widzę samochód.

Nie widzę samochodu.

📌 2. Rekcja czasownika

Польский глагол требует конкретного падежа.

Это не переводится логикой русского языка — это запоминается как модель.

Например:

Szukam pracy.

(НЕ: szukam pracę ❌)

Pomagam mamie.

(НЕ: pomagam mamę ❌)

Interesuję się sportem.

(НЕ: interesuję się sport ❌)

📌 3. Najczęstsze pułapki dla rosyjskojęzycznych

❌ Boję się egzamin

✅ Boję się egzaminu

❌ Dziękuję ciebie

✅ Dziękuję tobie

❌ Rozmawiam z kolega

✅ Rozmawiam z kolegą

Ćwiczenie 1

Uzupełnij w odpowiednim przypadku:

  1. Szukam (nowa praca)
  2. Nie mam (wolny czas)
  3. Potrzebuję (dobry lekarz)
  4. Słucham (polska muzyka)
odpowiedź
  1. Szukam nowej pracy ✔
    → dopełniacz: nowa praca → nowej pracy
  2. Nie mam wolnego czasu ✔
    → отрицание + dopełniacz
  3. Potrzebuję dobrego lekarza ✔
    → potrzebować + dopełniacz
  4. Słucham polskiej muzyki ✔
    → słuchać + dopełniacz
Ćwiczenie 2

Uzupełnij:

  1. Dziękuję (ty)
  2. Pomagam (mój brat)
  3. Gratuluję (koleżanka) sukcesu
  4. Wierzę (on)
Odpowiedź
  1. Dziękuję tobie.
    Можно ещё: Dziękuję ci (разговорная форма).
  2. Pomagam mojemu bratu.
    Celownik: mój brat → mojemu bratu
  3. Gratuluję koleżance sukcesu. koleżanka → celownik → koleżance (окончание -e, а не -i)
  4. Wierzę mu.
    jemu — используется для усиления, противопоставления: „Wierzę jemu, a nie tobie.”
Ćwiczenie 3

Uzupełnij:

  1. Interesuję się (psychologia)
  2. Rozmawiam z (szef)
  3. Jestem (nauczycielka)
  4. Zajmuję się (marketing)
Odpowiedź
  1. Interesuję się psychologią.
  2. Rozmawiam z szefem.
  3. Jestem nauczycielką.
  4. Zajmuję się marketingem.
Ćwiczenie 4

Popraw błędy:

  1. Szukam pracę.
  2. Dziękuję ciebie.
  3. Nie mam czas.
  4. Interesuję się sport.
Odpowiedź
  1. Szukam pracy.
  2. Dziękuję tobie.
    (в обычной речи чаще: Dziękuję ci, но твой вариант корректный)
  3. Nie mam czasu.
  4. Interesuję się sportem.
Ćwiczenie 5

Wybierz poprawną formę:

  1. Boję się (egzamin / egzaminu)
  2. Używam (telefon / telefonu)
  3. Pomagam (mamę / mamie)
  4. Rozmawiam z (kolega / kolegą)
Odpwiedź
  1. Boję się egzaminu.
  2. Używam telefonu.
  3. Pomagam mamie.
  4. Rozmawiam z kolegą.
Ćwiczenie 6

Przekształć na przeczenie:

  1. Mam pieniądze.
  2. Widzę film.
  3. Znam tego człowieka.
  4. Mam pytanie.
Odpwiedź
  1. Nie mam pieniędzy.

    Mianownik: pieniądze

    Dopełniacz (мн.ч.): pieniędzy

    Это форма нестандартная, её нужно просто запомнить.
  2. Nie widzę filmu.
  3. Nie znam tego człowieka.
  4. Nie mam pytania.
Ćwiczenie 7

Ułóż zdania z podanych wyrazów:

  1. szukać / ja / mieszkanie
  2. pomagać / ona / rodzice
  3. interesować się / my / sztuka
  4. dziękować / ja / nauczyciel
Odpowiedź
  1. Szukam mieszkania.

    все глаголы типа szukać, potrzebować, używać → dopełniacz.

    mieszkanie → mieszkania

  2. Pomaga rodzicom.
    rodzice → celownik → rodzicom
  3. Interesujemy się sztuką.
  4. Dziękuję nauczycielowi.
Ćwiczenie 8

Przetłumacz:

  1. Я ищу новую работу.
  2. Я боюсь экзамена.
  3. Я помогаю маме.
  4. Я интересуюсь историей.
Odpowiedź
  1. Szukam nowej pracy.
  2. Boję się egzaminu.
  3. Pomagam mamie.
  4. Interesuję się historią.
    historia → narzędnik → historią
Ćwiczenie 9

Wstaw poprawną formę:

  1. Nie słucham (radio).
  2. Uczę się (język polski).
  3. Marzę o (wakacje).
  4. Zależy mi na (praca).
Odpowiedź
  1. Nie słucham radia.

    słuchać → dopełniacz

    radio → radia

  2. Uczę się języka polskiego.

    uczyć się → dopełniacz

    język → języka

    polski → polskiego

  3. Marzę o wakacjach.

    wakacje — это множественное число.

    После „o” → miejscownik.

    wakacje → wakacjach

  4. Zależy mi na pracy.

    praca → miejscownik → pracy

    -a → -y / -i (в miejscowniku)

Ćwiczenie 10

Uzupełnij dialog:

  1. Czego szukasz?
  2. Szukam ________.
  1. Komu pomagasz?
  2. Pomagam ________.
Odpowiedź

🔹 1️⃣ Szukam pracy.

szukać → dopełniacz → praca → pracy.


🔹 2️⃣ Pomagam córce.

córka → celownik → córce

📌 Важная грамматика

Celownik — самый слабый падеж у русскоговорящих.

Нужно помнить окончания:

🔹 Мужской род
  • brat → bratu
  • kolega → koledze
  • student → studentowi
  • nauczyciel → nauczycielowi
  • sąsiad → sąsiadowi
  • przyjaciel → przyjacielowi

Чаще всего: -owi (самый частый вариант)

🔹 Женский род
  • mama → mamie
  • koleżanka → koleżance
  • praca → pracy
  • kobieta → kobiecie
  • córka → córce

Обычно:

  • -a → -ie
  • -ka → -ce
  • -ga → -dze
Нестандартные формы множественного dopełniacza:
  • pieniądze → pieniędzy
  • ludzie → ludzi
  • dzieci → dzieci
  • przyjaciele → przyjaciół

✅ LEKCJA 1 — PODSUMOWANIE

🟢 Что уже стабильно
  • Dopełniacz (ед. число) — уверенно
  • Narzędnik — почти без ошибок
  • Отрицание + dopełniacz — понимаешь
  • Rekcja основных глаголов — в целом осознаёшь
🟡 Что требует доработки (зафиксировано)
  1. Celownik liczby mnogiej

    kolegom, rodzicom, kobietom — нужно автоматизировать

  2. Miejscownik (особенно множественное число)

    wakacjach, na pracy и т.п.

  3. Орфография сложных слов

    psychologia, przyjaciel, język itd.

  4. Нестандартные формы множественного dopełniacza

    pieniędzy, przyjaciół, ludzi
